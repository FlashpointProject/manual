I get an error that a component couldn't be downloaded while installing Flashpoint
Wait a couple minutes and then retry the download. If the error reappears, it is possible that the component server is either overloaded or down for maintenance; in that case, you can download a zipped copy of Flashpoint from the download page.


The Flashpoint window isn't appearing when I run it
  1. Right-click on the taskbar and select Task Manager.
  2. In the opened Task Manager window, go to the Details tab.
  3. Scroll down until you see one or more instances of Flashpoint.exe.
  4. Select them and then click the End Task button until all Flashpoint.exe instances are gone.

I get an error saying that ffmpeg.dll was not found
If you have the Avira antivirus installed, add an exception for your Flashpoint folder.

To restore the missing ffmpeg.dll file:
  1. Open Flashpoint Manager by navigating to the Manager folder in the Flashpoint directory and launching FlashpointManager.exe.
  2. Go to the Update Components tab and click the Install updates button if there are any pending updates. If the launcher was one of the pending updates, you do not need to follow the remaining steps.
  3. Go to the Settings tab and click the Check for missing files button. The manager will then scan for missing files and the launcher should appear in the list.
  4. Click the Redownload components button to redownload the launcher component, and with it the missing ffmpeg.dll file.

Launching a game or animation results in a blank window
If you originally downloaded Flashpoint using the zipped copy, make sure that the prerequisites listed on the download page are installed.

If they are installed and entries are still opening a blank window, it is likely the result of antivirus interference. Follow this guide to add an antivirus exception for Flashpoint.
