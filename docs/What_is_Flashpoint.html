What is Flashpoint?
BlueMaxima's Flashpoint is a community-run archive of browser games and animations. Its official website is bluemaxima.org/flashpoint.

The only official download page for Flashpoint is bluemaxima.org/flashpoint/downloads. If you installed Flashpoint from any other source (Uptodown, Softonic, Techspot, etc.), we cannot guarantee that your copy is the latest version or has not been modified for malicious purposes.
