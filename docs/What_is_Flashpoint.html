What is Flashpoint?
BlueMaxima's Flashpoint is a community-run archive of browser games and animations. Its official website is bluemaxima.org/flashpoint.

The only official download page for Flashpoint is bluemaxima.org/flashpoint/downloads. If you installed Flashpoint from any other source (except the official Internet Archive mirrors by BlueMaxima and dri0m), we cannot guarantee that your copy is the latest version and has not been modified in some way.
